# 个性化在线编程与答疑系统可执行开发计划

## 1. 计划说明

本计划基于当前项目代码结构与已完成功能制定，目标是在**不改变基础技术栈**的前提下，按照“先稳核心链路，再提性能与可维护性，最后补学习闭环和工程能力”的顺序推进优化。

基础技术栈保持不变：

- 前端：Vue 3、Vite、Element Plus、Pinia、Vue Router、Monaco Editor
- 后端：Spring Boot、MyBatis、MySQL、Docker

允许按需增加的增强技术：

- 后端：Spring Cache、Redis、Resilience4j、Micrometer、JUnit 5、Testcontainers、MapStruct
- 前端：TypeScript 渐进引入、Vitest、Playwright、VueUse、ECharts
- 运维：Docker Compose、Nginx、Prometheus、Grafana

## 2. 总体实施顺序

建议按以下顺序推进：

1. 判题与代码运行稳定性治理
2. 提交记录、题库列表与高频查询优化
3. 前端大页面拆分与状态管理整理
4. 学习中心与个性化能力闭环补齐
5. 缓存、权限、安全与可观测性增强
6. 自动化测试与部署完善
