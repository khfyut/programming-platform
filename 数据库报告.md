## 4. 数据关联分析

### 4.1 题目与知识点关联
```sql
SELECT 
    COUNT(*) as total_problems,
    COUNT(DISTINCT pkp.problem_id) as problems_with_knowledge_points
FROM problem p
LEFT JOIN problem_knowledge_point pkp ON p.id = pkp.problem_id;
```

### 4.2 题目与关卡关联
```sql
SELECT 
    COUNT(*) as total_problems,
    COUNT(chapter_id) as problems_with_chapter,
    COUNT(level_id) as problems_with_level,
    ROUND(COUNT(chapter_id) * 100.0 / COUNT(*), 2) as chapter_coverage,
    ROUND(COUNT(level_id) * 100.0 / COUNT(*), 2) as level_coverage
FROM problem;
```

### 4.3 关卡与题目关联
```sql
SELECT 
    level_id,
    COUNT(*) as problem_count
FROM path_level_problem
GROUP BY level_id
ORDER BY problem_count DESC
LIMIT 10;
```

## 5. 数据质量分析

### 5.1 题目数据完整性
```sql
SELECT 
    COUNT(*) as total_problems,
    COUNT(CASE WHEN tags IS NOT NULL AND tags != '' THEN 1 END) as problems_with_tags,
    COUNT(CASE WHEN knowledge_points IS NOT NULL AND knowledge_points != '' THEN 1 END) as problems_with_knowledge_points,
    COUNT(CASE WHEN hints IS NOT NULL AND hints != '' THEN 1 END) as problems_with_hints,
    COUNT(DISTINCT p.id) as problems_with_testcases
FROM problem p
LEFT JOIN test_case tc ON p.id = tc.problem_id;
```

### 5.2 学习路径数据完整性
```sql
SELECT 
    COUNT(*) as total_paths,
    COUNT(CASE WHEN name IS NOT NULL AND name != '' THEN 1 END) as paths_with_name,
    COUNT(CASE WHEN language IS NOT NULL AND language != '' THEN 1 END) as paths_with_language,
    COUNT(CASE WHEN direction IS NOT NULL AND direction != '' THEN 1 END) as paths_with_direction
FROM learning_path;
```
