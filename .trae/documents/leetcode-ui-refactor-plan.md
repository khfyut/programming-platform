# UI重构计划 - 力扣风格在线编程平台

## 项目概述
将当前后台管理平台风格的UI重构为力扣风格的在线编程平台界面,保留登录页供管理员使用。

## 设计理念
- **简洁专业**: 采用力扣的简洁设计风格,突出核心功能
- **三栏布局**: 题目详情页采用经典的三栏布局(题目面板 + 编辑器 + 结果面板)
- **卡片式设计**: 使用卡片式组件提升视觉层次
- **力扣标准**: 难度标签、颜色方案、交互方式参考力扣标准
- **响应式设计**: 支持桌面端和移动端

## 重构范围

### 1. 主布局 (MainLayout.vue)
**当前状态**: 后台管理平台风格,侧边栏 + 顶部导航
**重构目标**: 力扣风格顶部导航栏

**改动点**:
- 移除侧边栏,改为顶部导航栏
- 导航项: 首页、题库、竞赛、社区、题解、我的
- 搜索框集成在顶部导航栏
- 用户头像和下拉菜单在右上角
- 简洁的logo设计
- 移除面包屑导航

**设计特点**:
- 固定顶部导航栏(高度60px)
- 白色背景 + 底部边框
- 导航项悬停效果
- 搜索框带搜索图标

---

### 2. 题目列表页 (Problems.vue)
**当前状态**: 表格形式,筛选栏 + 表格 + 分页
**重构目标**: 力扣风格题目列表

**改动点**:
- 简化筛选栏,只保留难度和标签筛选
- 表格改为卡片列表或优化表格样式
- 难度标签采用力扣标准(简单-绿色、中等-黄色、困难-红色)
- 题目标题可点击跳转
- 添加题目标签(数组、字符串、动态规划等)
- 优化分页器样式

**设计特点**:
- 卡片式题目展示
- 悬停效果
- 难度标签渐变背景
- 题目标题蓝色链接样式

---

### 3. 题目详情页 (ProblemDetail.vue)
**当前状态**: 三栏布局,但样式偏后台管理
**重构目标**: 力扣风格三栏布局

**改动点**:

- 右侧结果面板(25%): 运行结果、测试用例、通过率
- 优化分割线样式和拖拽交互
- 简化Tab标签
- 优化按钮样式(运行-绿色、提交-蓝色)

**设计特点**:
- 经典三栏布局
- 可拖拽调整面板宽度
- 简洁的Tab样式
- 渐变按钮
- 测试用例卡片展示

## 技术实现

### CSS变量系统
基于现有的enhanced-theme.css,优化力扣风格变量:

```css
/* 力扣风格颜色 */
--leetcode-primary: #0066FF;
--leetcode-success: #00C853;
--leetcode-warning: #FFB300;
--leetcode-danger: #EE4D2E;
--leetcode-bg: #FFFFFF;
--leetcode-bg-secondary: #F7F8FA;
--leetcode-border: #E5E7EB;
--leetcode-text: #24292F;
--leetcode-text-secondary: #6B7280;
```

- **难度标签**: 简单(绿色)、中等(黄色)、困难(红色)

- **卡片**: 白色背景 + 边框 + 阴影
- **输入框**: 简洁边框 + 聚焦效果

### 动画效果
- 页面加载淡入
- 悬停上移效果
- 按钮点击反馈
- Tab切换动画

2. 优化响应式设计

## 注意事项

1. **保留功能**: 所有现有功能保持不变,仅优化UI
2. **兼容性**: 确保Element Plus组件正常工作
3. **响应式**: 支持桌面端(>768px)和移动端(<768px)
4. **性能**: 避免过度动画影响性能
5. **可维护性**: 保持代码结构清晰,便于后续维护
